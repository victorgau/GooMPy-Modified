# GooMPy-Modified
Interactive OSM Maps for Python

Modified from [GooMPy](https://github.com/simondlevy/GooMPy) by Simon D. Levy.

Replace Google Static Maps API with our own OSM module.

Run:

```shell
python example.py
```
